
# nvim

My Neovim Config file

## Requirement

- neovim version >= 0.8
- tar & gzip(Only For Linux)
- 7zip(Only For Windows)
- curl, wget
- git
- gcc, clang, or MinGW(Only For Windows)  and libstdc++
- Microsoft C++ Build Tools(Only For Windows)
- python
- pynvim (pip packages)
- nodejs & npm
- neovim (npm packages)
- cmake
- ripgrep & fd & fzf
